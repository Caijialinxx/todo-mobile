# To Do List in Mobile
一个轻量的在线待办清单。

## 技术栈
React-Native
关键词：`react-navigation` `Animated` `Keyboard` `KeyboardAvoidingView` `Touchable`

## 功能页
- 主页
    - 添加待办
    - 删除待办
    - 标记待办的完成状态
    - 跳转个人页
    - 当地时间的读取、导航栏的收缩
- 个人页
    - 返回主页
    - 注销账号
- 登录/注册页
    - 登录
    - 忘记密码
    - 注册
